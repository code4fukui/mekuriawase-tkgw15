# Mekuriawase TKG-15 (徳川15代めくりあわせ)

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A memory match game based on the 15 Shoguns of the Tokugawa dynasty. Test your knowledge of Japanese history by matching each shogun's name to their corresponding generation number.

## Demo

**Play the game here: https://github.com/code4fukui/mekuriawase-tkgw15

## Gameplay

- **Objective:** Match all 15 Tokugawa Shoguns with their correct generation number.
- **How it works:** Click on the face-down cards to reveal them. One card in a pair shows the Shogun's name and portrait, while the other shows their generation number (e.g., "1st Shogun") and the same portrait.
- **Winning:** Once all pairs are matched, your total time is displayed.
- **Replay:** Use the "Retry" button to shuffle the cards and play again.

## Data Source

This game uses portrait images and data from the [Tokugawa 15 Shoguns Open Data](https://code4fukui.github.io/tokugawa15/) project.
