# Mekuriawase TKG-15 (徳川15代めくりあわせ)

徳川15代将軍を題材とした神経衰弱ゲームです。将軍の名前と対応する代数を一致させて、日本史の知識を試してみましょう。

## デモ

**ゲームはこちらで遊べます: https://code4fukui.github.io/mekuriawase-tkgw15/**

## 遊び方

- **目的:** 徳川15代将軍すべてについて、名前と正しい代数を一致させること。
- **仕組み:** 裏向きのカードをクリックして表に返します。ペアとなるカードの一方には将軍の名前と肖像が、もう一方には同じ肖像と代数（例：「1代将軍」）が表示されています。
- **クリア条件:** すべてのペアを一致させると、クリアタイムが表示されます。
- **リトライ:** 「Retry」ボタンを押してカードをシャッフルし、再挑戦できます。

## データソース

このゲームは、[Tokugawa 15 Shoguns Open Data](https://code4fukui.github.io/tokugawa15/)プロジェクトの肖像画像とデータを使用しています。

## クレジット

このプロジェクトは、福野泰介氏が作成したオープンデータ神経衰弱ゲーム[めくりあわせ (Mekuriawase)](https://fukuno.jig.jp/2014/mekuriawaseodp)のフォークです。
