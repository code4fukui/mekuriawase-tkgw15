# 徳川15代めくりあわせ

徳川15代将軍の肖像を使った記憶力ゲームです。対になった絵札を見つけると、クリアタイムが表示されます。

## デモ
https://code4fukui.github.io/mekuriawase-tkgw15/

## 機能
- 徳川15代将軍の肖像を使っためくりあわせゲーム
- 対になった絵札を見つけるとクリアタイムが表示される
- リトライボタンで再プレイできる

## データ・API
- 使用データ: [徳川15代オープンデータ](https://code4fukui.github.io/tokugawa15/)
- フォーク元: [めくりあわせ - 観光オープンデータを使ったアプリ(ODP)](https://fukuno.jig.jp/2014/mekuriawaseodp)
